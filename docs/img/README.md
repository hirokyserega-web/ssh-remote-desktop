# Скриншоты для README

В этой папке лежат **плейсхолдеры** — `client.png` и `server-gui.png` — на которые
ссылается `README.md` / `README.en.md`. Их нужно заменить на реальные снимки экрана.

## Что нужно

1. **`client.png`** — снимок окна `rd-client` с открытой сессией
   (рабочий стол удалённой Linux-машины + тулбар). Желательно 1280×720 или больше.
2. **`server-gui.png`** — снимок окна `rd-server-gui` (форма настроек + блок
   «Управление сервером» со статусом + хвост лога). Желательно ~720×800.

## Как снять

```bash
# Клиент: подключитесь к серверу, дождитесь рабочего стола, сделайте снимок окна
rd-client --host 127.0.0.1 --user $USER --key-path ~/.ssh/id_ed25519

# Панель сервера: запустите GUI и сделайте снимок окна
rd-server-gui
```

Под Linux снимок активного окна — `gnome-screenshot -w`, `scrot -u`, `spectacle -a`
или `grim -g "$(slurp)"` (Wayland). Под Windows — `Win+Shift+S`.
